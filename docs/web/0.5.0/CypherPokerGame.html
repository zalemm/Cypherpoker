Class: CypherPokerGame

CypherPokerGame(cypherpokerRef, tableObj, playerInfoopt, ContractClassopt)

Manages game logic, CypherPokerPlayer, CypherPokerContract, and CypherPokerAnalyzer instances, and other game-specific properties for a single CypherPoker.JS Texas Hold'em game (hand).
