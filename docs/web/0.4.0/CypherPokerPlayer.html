(readonly) keychain

Properties:
Name Type Description
keychain Array Indexed array of keypair objects used by the player. These are stored in ascending order with index 0 being the newest keypair. For other players in a game, this array will typically remain empty until the end of the game (verification). After a game ends this array should be archived if needed as it is usually reset at the start of a new game (hand).
